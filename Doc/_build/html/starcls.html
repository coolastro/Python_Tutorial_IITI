The Star Class¶

class PlSys_Classes.Star(massin, radiusin)¶

This class builds the framework to create a Star object with certain mass and radius.

Class to create objects related stars.

Args:

massin (float) : Stellar Mass

radiusin (float) : Radius of the star

static get_spectral_types()¶

The static method that is common to all stars and defines the spectral type of the star.

get_surface_gravity()¶

Computes the surface gravity of the star from the mass and radius.
